Version 0.7.0 - Snapshots

Username and password

Live images have two users: root and liveuser both without password.

At the graphical login prompt just hit Enter to log in.

Hardware Support

Hardware supported by Open Source drivers should work. Here's the list of supported GPUs:

  • Intel: i915 or newer cards
  • AMD/ATI: using the open source driver except for cards supported by radeonsi
  • NVIDIA: cards supported by the nouveau open source driver
  • VMware
  • QEMU with QXL

VirtualBox is not currently supported due to lack of drivers for Wayland.
